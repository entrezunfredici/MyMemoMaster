Cette page reference les vues declarees dans `src/router/routes.js` et les ecrans presents dans `src/pages`.

## Lecture rapide

- `Apercu interactif` signifie qu'une story charge la vue dans son contexte applicatif.
- `Documentation seule` signifie que la vue est referencee, mais pas previsualisee pour eviter un rendu trompeur sans backend ou mocks assez solides.
- Les routes privees sont documentees avec un contexte d'authentification simule dans Storybook.
